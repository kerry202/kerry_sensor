此APP大体实现的功能如下：

1、支持手机传感器数量探测，支持加速度，磁场，距离等传感器数据的实时获取

2、使用了一些github上的第三方控件来美化UI，

3、支持全屏显示

4、通过旧版的xUtils 2实现了数据库实时记录

5、通过POI来将数据库的数据导出为excel，方便以后分析
